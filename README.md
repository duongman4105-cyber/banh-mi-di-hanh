# Bánh Mì Dì Hạnh - Website

Đây là mã nguồn website giới thiệu quán **Bánh Mì Dì Hạnh** tại Chợ Long Phước. Website được thiết kế lại hoàn toàn với phong cách hiện đại, ấm cúng, tập trung vào trải nghiệm ẩm thực đường phố Việt Nam.

## 🌟 Tính Năng Nổi Bật

*   **Giao diện Responsive:** Tương thích hoàn hảo trên mọi thiết bị (Máy tính, Tablet, Điện thoại).
*   **Thiết kế độc đáo:**
    *   Sử dụng tông màu **Cam Đất (#d35400)** và **Kem (#fffbf0)** kích thích vị giác.
    *   Font chữ **Amatic SC** tạo cảm giác thủ công, thân thiện.
*   **Các phần chính:**
    *   **Trang chủ (Hero):** Banner ấn tượng với lời chào mời hấp dẫn.
    *   **Câu chuyện (About):** Giới thiệu về lịch sử hơn 20 năm và tâm huyết của Dì Hạnh.
    *   **Thực đơn (Menu):** Hiển thị các món Best Seller, Must Try với hình ảnh bắt mắt.
    *   **Tại sao chọn (Why Us):** Nhấn mạnh vào An toàn, Hương vị và Tốc độ.
    *   **Thống kê (Stats):** Các con số ấn tượng về quán.
    *   **Liên hệ (Contact):** Tích hợp bản đồ Google Maps, thông tin liên hệ và giờ mở cửa.

## 🛠️ Công Nghệ Sử Dụng

*   **HTML5 / CSS3 / JavaScript**
*   **Bootstrap 5.3.3:** Framework CSS chủ đạo.
*   **Thư viện hỗ trợ:**
    *   *AOS (Animate On Scroll):* Hiệu ứng khi cuộn trang.
    *   *GLightbox:* Hiển thị ảnh/video dạng popup.
    *   *Swiper:* Slider mượt mà cho phần đánh giá khách hàng.
    *   *PureCounter:* Hiệu ứng chạy số liệu thống kê.

## 🚀 Hướng Dẫn Cài Đặt & Chạy

Dự án này là **Static Website** (Web tĩnh), rất đơn giản để chạy:

1.  **Tải mã nguồn** về máy tính.
2.  Đảm bảo cấu trúc thư mục được giữ nguyên (đặc biệt là thư mục `assets`).
3.  Nhấp đúp vào file `index.html` để mở trên trình duyệt web (Chrome, Edge, Firefox, Safari...).

## 📂 Cấu Trúc Thư Mục

```text
Yummy/
├── assets/              # Chứa tài nguyên tĩnh
│   ├── css/             # File CSS chính (main.css)
│   ├── img/             # Hình ảnh (món ăn, đầu bếp, background...)
│   ├── js/              # File JavaScript xử lý logic (main.js)
│   └── vendor/          # Các thư viện bên thứ 3 (Bootstrap, AOS...)
├── forms/               # Các file xử lý form (PHP)
├── index.html           # Trang chủ (File chính)
└── README.md            # Tài liệu hướng dẫn
```

## 🎨 Tùy Chỉnh Giao Diện

Các đoạn mã CSS tùy chỉnh cho giao diện mới (Màu sắc, Font chữ, Card style) được đặt trong thẻ `<style>` tại phần `<head>` của file `index.html` để dễ dàng chỉnh sửa nhanh.

*   **Màu chủ đạo:** `--color-primary: #d35400`
*   **Màu nền:** `--color-bg: #fffbf0`
*   **Tùy biến & Thiết kế lại:** Dành riêng cho thương hiệu Bánh Mì Dì Hạnh.
